Exceptions
================


``UserNotFound``
^^^^^^^^^^^^^^^^^^^^^^

`This Exception is raised when the username passed to Twitter Object is not Found`


``GuestTokenNotFound``
^^^^^^^^^^^^^^^^^^^^^^

`This Exception is raised when the module is unable to get the guest token from Twitter`

``InvalidTweetIdentifier``
^^^^^^^^^^^^^^^^^^^^^^

`This Exception is raised when getting the standalone tweet detail and the tweet identifier is invalid`

``ProxyParseError``
^^^^^^^^^^^^^^^^^^^^^^

`This Exception is raised when there is an error in proxy format`

``UserProtected``
^^^^^^^^^^^^^^^^^^^^^^

`This Exception is raised when the username passed to Twitter Object is has profile visible only to its followers`

``UnknownError``
^^^^^^^^^^^^^^^^^^^^^^

`This Exception is raised when the error occurs which is unknown to the module`
