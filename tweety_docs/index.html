Welcome to Tweety¶

Welcome to Tweety’s documentation. Get started with Dependencies and then get an overview with the Quickstart. The rest of the docs describe each component of Tweety in detail, with a full reference in the All Available Functions section.

Twitter’s API is annoying to work with, and has lots of limitations — luckily their frontend (JavaScript) has it’s own API, which I reverse–engineered. No API rate limits. No restrictions. Extremely fast.
